# DecimScanner

## Overview  
A python package for threaded scanning

## Features  

<ins>TCP Scans</ins>  
* TCP Connect  
* SYN/Stealth Scan  
* FIN Scan  
* NULL Scan  
* ACK Scan  
* XMAS Scan  
* Window Scan  
* Idle Scan  

<ins>UDP Scans</ins>    
* UDP Connect

<ins> ICMP Scans</ins>  
* ICMP Ping
* IP Protocol Scan
